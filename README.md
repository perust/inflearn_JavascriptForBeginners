# inflearn_JavascriptForBeginners

인프런 코딩앙마 **왕초보 자바스크립트** 강의를 들으며 정리한 입문용 JavaScript 학습 코드입니다. 변수, 자료형, 연산자, 조건문, 반복문, 함수, 객체, 배열까지 기본 문법을 강의 단위 HTML 파일로 정리했습니다.

## 강의 정보

- 플랫폼: Inflearn
- 강의: 코딩앙마의 왕초보 자바스크립트
- 시작일: 2022.03.01

## 구성

```text
.
├── index.html
├── 1_변수.html

├── 3_alert, prompt, confirm.html
├── 4_형변환.html
├── 5_기본 연산자.html
├── 6_비교 연산자,조건문(if,else).html
├── 7_논리 연산자.html
├── 8_반복문.html
├── 9_switch문.html
├── 10_function.html

├── 12_객체.html
├── 13_객체-method,this.html
└── 14_배열.html
```

## 실행 방법

브라우저에서 원하는 HTML 파일을 직접 열면 됩니다.

```bash
open index.html
```

로컬 서버를 사용할 수도 있습니다.

```bash
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`에 접속합니다.

## 학습 주제

- 변수와 자료형
- `alert`, `prompt`, `confirm`
- 형변환
- 기본/비교/논리 연산자
- 조건문과 반복문
- `switch`
- 함수 선언식, 함수 표현식, 화살표 함수
- 객체, 메서드, `this`
- 배열
